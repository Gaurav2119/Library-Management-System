To run the book buddy web app

Step 1: Setup backend
	-> navigate to package manager console and run 'update-database' command
	-> run the backend

Step 2: Setup Frontend
	-> install node modules using 'npm install' command in terminal
	-> run the frontend using 'ng serve' command in terminal
